## Getting Started

First, install packages then run the development server:

```bash
npm install
#or
yarn install
#or
pnpm install

---------------

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
